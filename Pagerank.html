Hitung Manual#

Iterasi 0 dan 1#

untuk iterasi ke 0 akan menggunkan nilai 0.111111 yang berasal dari 1/banyaknya Node, pada gambar graph diatas node yang digunakan ada 9 jadi 1/9 = 0.111111

outdeggree dari 9 node ini sebanyak:

  • Node 1: 2

  • Node 2: 2

  • Node 3: 2

  • Node 4: 2

  • Node 5: 1

  • Node 6: 1

  • Node 7: 1

  • Node 8: 1

  • Node 9: 1

Lalu untuk iterasi ke 1

  • P1 = P2/outP2
    P1 = 0.111111/2 = 0.055556

  • P2 = P1/OutP1
    P2 = 0.111111/2 = 0.055556

  • P3 = P1/OutP1
    P3 = 0.111111/2 = 0.055556

  • P4 = P1/OutP1+P9/OutP9
    P4 = 0.111111/2+0.111111/1 = 0.166666

  • P5 = P2/OutP2+P3/OutP3
    P5 = 0.111111/2+0.111111/2 = 0.111111

  • P6 = P2/OutP2+P5/OutP5
    P6 = 0.111111/2+0.111111/1 = 0.166666

  • P7 = P3/OutP3+P4/OutP4+P6/OutP6
    P7 = 0.111111/2+0.111111/2+0.111111/1 = 0.166666

  • P8 = P4/OutP4+P7/OutP7
    P8 = 0.111111/2+0.111111/1 = 0.166666

  • P9 = P8/OutP8
    P2 = 0.111111/1 = 0.111111
